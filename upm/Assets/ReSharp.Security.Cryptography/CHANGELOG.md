# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).



## [1.1.0] - 2026-04-02

### Added

- Adds some extended methods for different cryptographic service.



## [1.0.0] - 2026-04-01

### Added

- Adds class **AesCryptoUtility** to provide AES cryptographic service.
- Adds class **CryptoUtility** to provide common cryptographic service.
- Adds class **Md5CryptoUtility** to provide MD5 hash cryptographic service.
- Adds class **RsaCryptoUtility** to provide RSA cryptographic service.
- Adds class **ShaCryptoUtility** to provide SHA hash cryptographic service.
- Adds class **XxteaCryptoUtility** to provide XXTEA cryptographic service.
